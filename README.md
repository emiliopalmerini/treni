# Treni

Train tracking web application built with Go, HTMX, and templ.

## Quick Start

```bash
make run
```

The server will start on the configured address (default: `:8080`).

## Build

```bash
make build
```

## Test

```bash
make test
```

## Tech Stack

- **Go** - Backend
- **templ** - Type-safe HTML templates
- **HTMX** - Frontend interactivity
- **SQLite** - Database
- **sqlc** - Type-safe SQL
